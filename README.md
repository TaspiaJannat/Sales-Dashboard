An interactive and data-driven **Excel Dashboard** designed to analyze sales team performance, call efficiency, and revenue trends. This project provides actionable insights to help businesses track agent productivity and regional sales growth.

---

## üöÄ Key Features

This dashboard provides a comprehensive 360-degree view of sales operations:

* **Executive Summary (KPIs):** Real-time tracking of 4 core metrics:
    * **Total Calls:** 18,300
    * **Calls Reached:** 3,619
    * **Total Deals Closed:** $706,194.89
    * **Average Deal Value ($):** 1,309
* **Agent Performance Tracker:** A detailed breakdown of each agent's total calls vs. successful reaches and closed deals.
* **Monthly Deal Trends:** A horizontal bar chart visualizing "Deal Value By Month" (e.g., March recorded the highest sales at $60,127).
* **Geographic Sales Map:** A visual representation of revenue distribution across different regions of Bangladesh.
* **Interactive Slicers:** Dynamic filters for "Sales Agent Name" allowing users to drill down into specific individual performance instantly.
